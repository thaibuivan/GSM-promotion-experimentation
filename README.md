# GSM Promotion Experimentation

## Project Overview
This project aims to analyze NYC Taxi data to identify optimal rider segments for a ride-hailing promotion strategy.

## Repository Structure
- `data/`: Raw and processed datasets (ignored in Git).
- `docs/`: Documentation, data dictionaries, and preprocessing reports.
- `notebooks/`: Jupyter notebooks separated by analytical weeks.
- `reports/`: Presentation decks and feature catalogs.
